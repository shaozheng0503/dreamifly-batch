# Model Selection

- Cheap anime batches: `Wai-SDXL-V150` or `Wai-SDXL-V170`
- Fast Chinese/general text-to-image: `Z-Image-Turbo`
- Image editing with references: `Qwen-Image-Edit` with `img=`
- High-quality or complex images: `gpt-image-2`; use `nano-banana-2` only with explicit cost approval
- One source image to video: `Wan2.2-I2V-Lightning`
- Text-to-video or multi-reference video: `happyhorse-1.0`

Cost guardrail: video models and `nano-banana-2` are expensive. Any item estimated at 5+ credits must be confirmed by the user before running a real API call.
